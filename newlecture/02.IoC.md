# 1. IoC
- Inversion Of Control
- 역순으로 객체를 생성

##  1.1. 예시
```
주문서

메인 보드 : A사 제품
VGA 카드 : B사 제품
Memory : S사 16G
CPU : I사 제품
```
1. 컴퓨터를 조립할 때 필요한 부품들을 모아 조립해야 하는데, <br>
소프트웨어는 xml, annotation을 이용하여 주문서를 작성할 수 있다.
2. 컨테이너 안에 부품들을 조립하여 담아줄 수도 있다.

- 일반적인 일체형
> A -> B -> C -> D <br>
> ───────>

- 결합형(역순으로 객체를 생성)
> A -> B -> C -> D <br>
> <───────
