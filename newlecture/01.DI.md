# 3. DI(Dependency Injection)
- Composition(일체형) has a
```java
class A {
    private B b;
    public A() {
        b = new B();
    }
}
```
- Association(조립형) has a
```java
class A {
    private B b;
    public A() {
        // 객체를 직접 생성하지 않는다.
    }

    public void setB(B b) {
        this.b = b;
    }
}
```

## 3.1. 주입 방법
1. Setter Injection(setter)
```java
B b = new B();
A a = new A();

a.setB(b);

```
2. Construction Injection(생성자)
```java
B b = new B();
A a = new A(b);
```
